CaseLawg

Built in collaboration with a fellow General Assembly cohort member and a former attorney, CaseLawg is a comprehensive case management system designed to help legal professionals streamline their workflow and focus on winning cases.

Technologies

  • Server-side: Django, Python
  • Database: PostgreSQL
  • Client-side: JavaScript, Bootstrap, HTML, CSS

Features

  • User Authentication: Secure login and sign-up powered by Django's built-in authentication system.
  • Case Management: Full CRUD functionality for creating, editing, and deleting cases.
  • Task Management: Easily create, update, and track tasks associated with specific cases.
  • Time Tracking: Track billable hours and maintain detailed records for client billing and court reports.

TaskRabbit

A dynamic task management application designed to streamline task organization and boost productivity. TaskRabbit empowers users to create, edit, view, and delete tasks effortlessly.

Technologies

  • Server-side: Node.js, Express.js
  • Client-side: HTML, CSS, JavaScript, EJS
  • Database: MongoDB, Mongoose

Features

  • User Authentication: Secure sign-up, sign-in, and password reset functionality, ensuring user data is protected.
  • Comprehensive Task Management: Full CRUD functionality that allows users to manage tasks with ease.
  • Advanced Task Organization: Task categorization and prioritization features that enable users to prioritize important tasks.
  • Responsive Design: Fully responsive UI that delivers a seamless experience across all browsers.

Crowdfunly

Crowdfunly is a full-stack web application that enables users to create and donate to crowdfunding campaigns. Built in collaboration with two fellow General Assembly cohort members, Crowdfunly provides a seamless experience.

Technologies

  • Server-side: Node.js, Express.js
  • Client-side: React.js
  • Database: MongoDB, Mongoose
  • Payment Processing: Stripe

Features

  • User Authentication: Secure sign-up, log-in, and account management powered by JWT and Bcrypt.
  • Campaign Management: Full CRUD functionality for creating, editing, viewing, and deleting crowdfunding campaigns.
  • Payment Integration: Integrated Stripe payment processing for secure donations.
  • Responsive Design: Fully responsive UI that delivers a seamless experience across all browsers.
  • Deployment: Frontend deployed on Netlify, backend deployed on Heroku.
